## Soccer Player and Object Detection

- Used RoboFlow Dataset - https://universe.roboflow.com/augmented-startups/football-player-detection-kucab
- Used YOLO26 and also YOLOv8n. the code contains only the model trained on the YOLO26 Framework
- Classes: player, football
- Epochs: 10
- Image size: 640
- mAP50:
  - all: 0.682
  - football: 0.398
  - player: 0.966
